# Proyect Backend - Coderhouse
Ecommerce proyect from the course of Backend in coderhouse

Proyect using mongoDB database

### Install and Run the Project

```sh
git clone https://github.com/julietacastano/proyecto-final
cd proyecto-final
npm install
npm start
```
